HTML and Javascript!

Hey it looks like you'd enjoy HTML and Javascript! HTML is the language used to make basic webpages on the web. Using HTML, CSS to style the HTML, and Javascript to make it dynamic, you can create beatiful and intuitive webpages. This is what is called "front-end development", which just refers to the parts of a website that the user interacts with.
