HTML and Javascript!

It looks like you'd enjoy HTML and Javascript! HTML is the language used to make basic webpages on the web. Using HTML, CSS to style the HTML, and Javascript to make it dynamic, you can create beatiful and intuitive webpages. This is what is called "front-end development", which just refers to the parts of a website that the user interacts with.

Doesn't sound like your thing?

Here are some other languages you can look into:
  • Java: A very versatile and widely used back-end language. It is specifically good if you want to make Android applications.
  • C#: A back-end language created by Microsoft. It's niche is mostly in large well-established companies.
  • PHP: Currently the most popular back-end development language. PHP is a great general language for just about any back end web work.
  • Ruby: An interactive language made for building web applications quickly and efficiently, using it's framework Rails. It is popular with smaller companies and startups.
